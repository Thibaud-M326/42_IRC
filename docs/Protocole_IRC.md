# IRC

Client :

dans un reseaux informatique le client est le logiciel que envoie des demandes ou des requetes a un serveur, il peut servir d’un programme manipule par une personne ou un bot. Il existe des clients de differents type : 

- HTTP (web)
- IRC (chat)
- FTP (transfert de donnee)

L'ordinateur client est généralement un [ordinateur personnel](https://fr.wikipedia.org/wiki/Ordinateur_personnel) ordinaire, se comportant dans ce contexte comme un [terminal](https://fr.wikipedia.org/wiki/Terminal_(informatique)) : il est équipé de logiciels relatifs aux différents types de demandes qui vont être envoyées, comme un [navigateur web](https://fr.wikipedia.org/wiki/Navigateur_web), un logiciel client pour le [World wide web](https://fr.wikipedia.org/wiki/World_Wide_Web_Consortium).

Serveur :

Un serveur informatique est un dispositif informatique (materiel ou logiciel) qui offre des services a un ou plusieurs clients (parfois des milliers). 

Les services les plus courant sont :

- L acces aux information sur le web
- le courier electronique
- partage de peripherique (disque dur, imprimante, etc..)
- Gestion de l’authentification et du controle d’acces

En fonctionnement, un serveur répond automatiquement à des requêtes 
provenant d'autres dispositifs informatiques (les clients), selon le 
principe dit client-server.

!!! le format des requettes et des resultats est normalise, en se conformant a des protocole reseaux !!!

Protocole Reseaux : 

Emsemble des protocole de communication visant a permettre la communication formate entre les clients et les serveur. 

Ce sont la couche de normalistation dont profite toutes les application en reseaux :

on trouve : 

- FTP (file transfert protocol)
- SSH
- HTTP
- SMTP (mail)
- IRC (chat)

IRC : Internet Relay Chat ([wikipedia : IRC](https://fr.wikipedia.org/wiki/Internet_Relay_Chat))

Protocole de communication textuel sur internet. Sert a la discussion instantatnee, sous la forme de groupe par l’intermediaire de canaux de discussions.

Il peut aussi etre utilise pour faire du transfert de fichier.

IRC - Histoire : 

Concu fin aout 1988, decrit initialement dans RFC 1459 par Jarkko Oikarinen (surnommé « WiZ ») et Darren Reed. Puis revise dans les RFC 2810 a RFC 2813.

A la fin des annee 1990, l’utilisation d’IRC diminue avec l’arrivee des messageries instantanee proprietaire comme MSN messenger.

IRC reste néanmoins utilisé dans certains milieux désirant se passer d'un programme client proprietaire.

IRC - Technique : 

IRC est un protoole utilisant TCP, l’utilisateur utilise un logiciel client IRC afin de se connecter a un des serveurs du reseaux,
